SUPERNOME DIVA 2 : contemplation du vide

Développement d'un logiciel pour le contrôle expressif de musique enregistrées



MonoReplay est un logiciel que j'ai conçu pour contrôler de manière expressive le déroulement d'une musique enregistrée. Chaque paramétre peut être contrôlé via le protocole OSC. Ainsi le musicien peut utiliser ses propres interfaces gestuelles pour contrôler le logiciel en temps réel.


Le musicien Nicolas Rivière a pris en main ce logiciel lors d'une performance filmée au sein de l'Institut Jean Le Rond d'Alembert le 10 novembre 2021.


Musique : Nicolas Rivière
Conception logicielle : Thomas Lucas
Lumière, cadre, montage : Damien Bestard
